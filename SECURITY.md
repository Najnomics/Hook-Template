# Security Policy

## Reporting
Report vulnerabilities privately to the maintainers before public disclosure.

## Scope
In-scope:
- `src/` contracts
- `script/` and `scripts/` deployment/demo logic
- dependency pinning/bootstrap logic

Out-of-scope:
- third-party upstream dependency vulnerabilities unless exploitable through this integration

## Notes
This repository provides reference implementations and controls for specialized market hooks, but no claim of complete attack resistance is made.
