# Overview

Hook Templates is a reusable Uniswap v4 template suite for specialized markets:
- Stablecoin markets
- RWA markets
- Long-tail token launches

The repo combines:
- production-style hook contracts
- deterministic dependency/deployment tooling
- demo scripts (local/testnet)
- tests across unit/edge/fuzz/integration
- a React launcher consuming shared ABIs/types

Use this project as a baseline to instantiate new specialized-market hooks with consistent safety patterns.
