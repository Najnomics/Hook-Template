# Templates

## Stablecoin Template
- Dynamic fee banding by peg-deviation proxy (tick distance).
- Volatility-spike fee elevation.
- Optional circuit-breaker-lite cooldown for extreme deviation.

## RWA Template
- Permissioned actor controls (allowlist).
- Session-window trading restrictions.
- Tick-jump and slippage guardrails for low-liquidity conditions.

## Long-Tail Template
- Launch-mode fee and max-trade constraints.
- Transition to normal mode by time or volume.
- Optional segmented order-flow cap per block.
